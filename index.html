mkcert: Your Tool For Quick Doc Genaretion

mkcert is a Python package to auto-genarete documents of all kinds, here you can finish that in a flash.

Installation

First of all, you'll need to certify if you have the Python properly installed on your PC. So, open an interactive shell and run:

$ python --version

Then, must return the version of Python you have installed, something like:

Python 3.10.6

Now, to download the mkcert package, open an interactive shell and run:

$ python<version> -m pip install mkcert
